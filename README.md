# mp1-hospital

## Client

My client is a hospital owner.

## Site Description

This project is a redesigned orthopedic clinic and hospital website. I reshaped the hospital's original English website, https://yshospital.net/en/, with a stronger focus on the hospital's International Healthcare Center. The site provides information about services, doctors, appointments, location, and international patient support.

## Live Site

https://youngseolee736.github.io/mp1-hospital/

## What I Learned

I learned how to use AI to help generate and improve an actual website. During this process, I also used JavaScript for the first time to add basic interactivity and dynamic content.
